# To Do List 프로젝트

나만의 To Do List를 만들 수 있습니다

## Services

- Task마다 label 색깔 지정 가능
- Task와 List Drag and Drog 가능

## Stacks

### Front-end

- `SCSS`
- `Javascript`
- `Pug`

### Back-end

- `Nodejs-Express`
- `MongoDB`
- `bcrypt`
